About the vulekamali datastore and API

This is a CKAN data portal that hosts the data on the vulekamali Budget Data Portal.

Use the vulekamali CKAN API to access the data in the datastore:

  • Structured Estimates of National Expenditure that you can explore with the OpenSpending API.
  • PDF documents and Excel spreadsheets that users can download and work with offline.

Structured Datasets

These datesets are available to download as Excel pivot tables and to query directly using the OpenSpending API.

Estimates of National Expenditure

Use the Estimates of National Expenditure (ENE) pivot tables and API to help users understand budget allocation to national departments.

  • ENE for 2018-19
  • ENE for 2017-18
  • ENE for 2016-17

Estimates of Provincial Expenditure

Use the Estimates of Provincial Revenue and Expenditure (EPRE) pivot tables and API to help users understand budget allocation to provincial departments.

  • ENE for 2017-18
  • ENE for 2016-17
  • ENE for 2015-16

What is vulekamali?

vulekamali is a partnership between National Treasury and Imali Yethu to make South Africa's National and Provincial budget data more accessible. Read more about vulekamali or watch the video.
