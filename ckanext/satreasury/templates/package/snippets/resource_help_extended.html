One dataset can have multiple resources

You can upload all the files that go together to help use and understand the data as multiple resources in the dataset.

For example, in a dataset about how government spending on a topic differs across the country, you might upload a PDF file which contains a report; an Excel Spreadsheet which contains tables in the report; a CSV file with the raw data for further analysis; a shapefile showing monitoring stations on the map; and any other data that others might find useful.
